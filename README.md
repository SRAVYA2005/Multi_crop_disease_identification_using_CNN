Developed a plant disease classification system using transfer learning with MobileNetV2. Trained models on
four crop datasets (apple, potato, tomato, and strawberry) individually and as a combined multi-crop model.
Achieved accuracies of 100%, 98%, 97%, and 100% for individual crop models and 98% accuracy for the
generalized multi-crop model. Validated the model using unseen leaf images (e.g., corn) to test its ability to
distinguish trained and untrained crops.
